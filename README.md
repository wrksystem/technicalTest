# Teste Técnico — Respostas

Descrição
---------

Repositório contendo as respostas e implementações fornecidas para um teste técnico.

Estrutura do repositório
------------------------

- `RESPOSTAS.md`: Contém as respostas das questões 01 e 02.(Markdiwn).
- `question03.dart`: Implementação em Dart da Questão 03.
- `question04.md`: Enunciado e/ou resposta da Questão 04 (Markdown).
- `question05.md`: Enunciado e/ou resposta da Questão 05 (Markdown).

Requisitos
----------

- Dart SDK instalado para executar o arquivo `question03.dart`.

Como executar / visualizar
--------------------------

- Executar o script Dart (`question03.dart`):

```bash
dart run question03.dart
```


Contato
-------

Para dúvidas ou correções, abra uma issue no repositório pelos meios dispostos no perfil.
